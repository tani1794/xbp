Reflection

1回目の講義の感想

今回の講義を通して、ホームページ制作は単にコードを書く作業ではなく、 情報をどう見せるかまで含めて考えるデザインと設計の積み重ねであることを強く実感しました。

気づいたこと

HTMLは要素を並べるだけではなく、内容の順序や見せ方によって受け取られ方が変わるため、 最初の構成づくりがとても重要だと感じました。

難しかったこと

センサーや抵抗（オーム）の配列が難しかった。ホームページのやり方がなかったら完成できなかったし、 アルディーノのを通してセンサーに明るさや暗さの値を設定してLEDライトの点灯を完成させる作業は難しくやり甲斐がありました。

今後に活かしたいこと

これからは、完成した見た目だけでなく、そこに至る設計の考え方も意識しながら、 伝わりやすさと表現の両方を高めていきたいです。

初回講義は、ページ制作の入口であると同時に、自分の中でデザインを感覚だけで捉えるのではなく、 構造と意図を持って形にしていく必要性を明確にした時間でした。

Motivation & Goals

これからの意欲・目標

今後は、ページを見た瞬間の印象だけでなく、内容の伝わり方まで設計できるようになり、 デザイン、設計、質の向上、意欲の向上を一つの流れとして積み上げていきたいと考えています。

短期目標

レイアウト、余白、文字の強弱を意識しながら、見やすく整ったページを安定して作れるようになる。

中期目標

コードを書く前に構成を考える習慣を身につけ、デザインと設計を結びつけたページ制作ができるようになる。

質の向上

毎回の制作で改善点を振り返り、見た目の完成度だけでなく、内容の伝わり方まで高めていくことを目指す。

意欲の向上

制作を重ねるたびに新しい工夫を取り入れ、自分の表現の幅を広げながら前向きに学び続けていく。

これから意識したい姿勢

ただ課題を提出するためではなく、自分の学びを作品として残す意識を持ち、丁寧に設計していく。

これからは、1回ごとの演習を点で終わらせず、デザインと設計を軸にしながら積み上げ、 表現の質を少しずつ高めていける自分を目標にしていきます。

Arduino Analysis

提出用の文章と3つのコード分類

以下では、Arduinoで扱った3つのプログラムを提出用の文章として整理し、そのあとに役割ごとに分類して説明しています。

今回の実習では、Arduinoを用いたLED制御とセンサー入力の基本について学んだ。1つ目のコードでは、13番ピンに接続したLEDを500ミリ秒ごとに点灯と消灯を繰り返させることで、デジタル出力の基礎を確認した。2つ目のコードでは、for文を使ってLEDを10回速く点滅させた後に3回ゆっくり点滅させることで、同じ出力でも繰り返し処理によって異なる発光パターンを作れることを理解した。3つ目のコードでは、アナログ入力から取得した値をシリアルモニタに表示し、センサーの値を数値として確認する方法を学んだ。これらのプログラムを通して、ArduinoではLEDのような出力機器の制御だけでなく、センサーからの入力を読み取って活用できることが分かり、入出力処理の基礎を理解することができた。

Output

1. LEDの基本点滅コード

digitalWrite()でLEDを点灯と消灯に切り替え、delay()で待ち時間を入れる基本的な出力制御のコードです。一定の周期で光らせることで、デジタル出力の基礎を理解できます。

  • pinMode(13, OUTPUT) で13番ピンを出力に設定する
  • HIGHで点灯、LOWで消灯する
  • delay(500) で0.5秒ごとの点滅になる
Loop

2. 繰り返しで変化をつけるコード

for文を使って点滅回数と速度を変えるコードです。10回の速い点滅と3回の遅い点滅を続けることで、単純なON/OFFよりも複雑な発光パターンを作れます。

  • for文で同じ処理を決まった回数だけ繰り返す
  • delay(300) と delay(2000) で速さを変えている
  • 点灯パターンの演出を学ぶコードである
Input

3. センサー値の読み取りコード

analogRead()でアナログ入力の値を取得し、Serial.println()でシリアルモニタに表示する入力処理のコードです。明るさセンサーや可変抵抗の値を数値として確認できます。

  • Serial.begin(9600) で通信速度を設定する
  • analogRead(1) でアナログ入力1番を読む
  • Serial.println(val) で値を画面に表示する

追加コード① LEDの基本点滅

void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(500);
  digitalWrite(13, LOW);
  delay(500);
}

追加コード② 点滅パターンの変化

void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  for (int i = 0; i < 10; i++) {
    digitalWrite(13, HIGH);
    delay(300);
    digitalWrite(13, LOW);
    delay(300);
  }

  for (int i = 0; i < 3; i++) {
    digitalWrite(13, HIGH);
    delay(2000);
    digitalWrite(13, LOW);
    delay(2000);
  }
}

追加コード③ センサー値の読み取り

void setup() {
  Serial.begin(9600);
}

void loop() {
  int val = 0;
  val = analogRead(1);
  Serial.println(val);
  delay(500);
}

この3つのコードは、Arduino学習の流れとして「出力の基本」「繰り返し処理による応用」「入力の取得」という順番で整理できる。これにより、LEDを光らせるだけでなく、センサーの値を取り入れながら制御を発展させられることを理解できた。
