Flask 講義まとめ

Pythonを使ってWebページを表示する仕組みを学ぶ

1. Flaskとは

Flaskとは、PythonでWebアプリケーションを作るための軽量なフレームワークである。 HTMLファイルを表示したり、フォームから送られたデータを受け取ったり、CSVファイルの内容をWebページに表示したりできる。

通常のHTMLだけでは、ページの見た目を作ることはできるが、データを読み込んだり、Pythonの処理結果を表示したりすることは難しい。 Flaskを使うことで、Pythonの処理とHTMLの表示をつなげることができる。

3. 最小のFlaskプログラム例

Flaskで最も基本的なプログラムは、トップページに文字を表示するものである。

app.py

from flask import Flask

app = Flask(__name__)

@app.route("/")
def index():
    return "FlaskでWebページを表示しました"

if __name__ == "__main__":
    app.run(debug=True)

このコードを実行し、ブラウザで http://127.0.0.1:5000 を開くと、 Flaskで作成したページが表示される。

4. HTMLを表示するFlaskプログラム例

実際のWebページでは、Pythonの文字列ではなく、HTMLファイルを表示することが多い。 その時に使うのが render_template である。

app.py

from flask import Flask, render_template

app = Flask(__name__)

@app.route("/")
def index():
    return render_template("index.html")

if __name__ == "__main__":
    app.run(debug=True)

templates/index.html

<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>Flask練習</title>
</head>
<body>
  <h1>FlaskでHTMLを表示しました</h1>
</body>
</html>
HTMLファイルは必ず templates フォルダの中に置く必要がある。

5. CSVファイルを読み込んで表示する例

今日の講義では、CSVファイルの内容をPythonで読み込み、それをHTMLに渡して表示する流れも重要である。

app.py

from flask import Flask, render_template
from csv import reader

app = Flask(__name__)

@app.route("/")
def index():
    with open("./mmcsv.csv", "r", encoding="utf-8") as csv_file:
        csv_reader = reader(csv_file)
        list_mm = list(csv_reader)

    return render_template("index.html", mmlist=list_mm)

if __name__ == "__main__":
    app.run(debug=True)

templates/index.html

<h1>CSVデータ一覧</h1>

<table border="1">
  {% for row in mmlist %}
    <tr>
      {% for item in row %}
        <td>{{ item }}</td>
      {% endfor %}
    </tr>
  {% endfor %}
</table>

6. GoogleマップのURLを作るプログラム例

CSVの中に緯度・経度のような位置情報が入っている場合、 その情報を使ってGoogleマップのURLを作ることができる。

app.py

from flask import Flask, render_template
from csv import reader

app = Flask(__name__)

@app.route("/")
def index():
    with open("./mmcsv.csv", "r", encoding="utf-8") as csv_file:
        csv_reader = reader(csv_file)
        list_mm = list(csv_reader)

    for data in list_mm:
        if len(data) > 3:
            data[3] = "https://www.google.com/maps?q=" + data[3].replace("(", "").replace(")", "").replace(" ", "")

    return render_template("index.html", mmlist=list_mm)

if __name__ == "__main__":
    app.run(debug=True)

templates/index.html

<h1>マップリンク一覧</h1>

{% for data in mmlist %}
  <div>
    <p>{{ data }}</p>
    <a href="{{ data[3] }}" target="_blank">Googleマップで見る</a>
  </div>
{% endfor %}
data[3] はCSVの4列目を表している。 Pythonでは数え方が0から始まるため、4列目は3番になる。

8. 今日の講義のまとめ

Flaskは、PythonとHTMLをつなぎ、Webページを動的に表示するための仕組みである。 HTMLだけでは固定されたページしか作れないが、Flaskを使うことで、 CSVファイルの読み込みや、データの加工、条件に応じた表示などが可能になる。

特に重要なのは、app.pyが処理を担当し、templates内のHTMLが表示を担当するという役割分担である。 この構造を理解することで、Pythonで作った情報をWebページに表示できるようになる。

今回の学習では、Flaskの基本、HTML表示、CSV読み込み、GoogleマップURLの作成などを通して、 Webアプリケーションの基礎的な流れを確認した。
